# 实验五 数组 
## 一、实验目的 
1. 掌握一维数组和二维数组的定义、赋值和输入输出的方法。 
2. 掌握字符数组和字符串函数的使用。 
3. 掌握与数组有关的算法（特别是排序算法）。 
 
## 二、实验准备 
1. 复习数组的基本知识。 
2. 复习字符串数组的特点和常用的字符串处理函数。
